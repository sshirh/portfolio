UPS Store AI Phone Receptionist

A production-ready AI-powered phone receptionist designed for a UPS Store–style retail environment. This system answers real phone calls, understands natural speech, and responds with accurate, business-aware information while maintaining a natural conversational flow.

View Source

Live Demo — AI Phone Receptionist

Demo showing a real inbound call, speech recognition, intent classification, and real-time responses with graceful handoff behavior.

Features

  • Natural-language voice interaction using Twilio Speech Recognition
  • Intelligent intent routing with OpenAI
  • Date-aware store hours (today, tomorrow, weekends, holidays)
  • Human handoff with live dialing during business hours
  • Graceful handling of after-hours calls
  • Professional conversational flow with confirmations and follow-ups
  • Safe handling of UPS tracking requests without collecting sensitive data
  • Clean fail-safes to prevent dropped or silent calls

Supported Caller Requests

  • Store hours (“When do you close tomorrow?”)
  • Location and directions
  • Notary services
  • Amazon returns
  • Printing and copying services
  • Packing and shipping services
  • Mailbox services
  • Package tracking guidance
  • Speak to a representative
  • End call / goodbye phrases

Technical Stack

  • Node.js
  • Express
  • Twilio Programmable Voice
  • OpenAI API
  • Luxon (timezone-safe date handling)
  • chrono-node (natural language date parsing)

Call Flow Overview

  1. Caller dials the store number
  2. AI receptionist greets the caller
  3. Speech is captured and transcribed by Twilio
  4. Intent is classified using keyword routing, then OpenAI if needed
  5. Context-aware response is generated
  6. Caller is prompted or connected to a representative
  7. Call ends cleanly on goodbye

Safety & Privacy

  • No tracking numbers are stored or logged
  • Callers are directed to official UPS tracking
  • Secrets stored in environment variables
  • .env excluded from version control
